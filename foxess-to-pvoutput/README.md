# Fox ESS Cloud to PVOutput exporter

Usage:

```
usage: main.py [-h] --foxess-user FOXESS_USER --foxess-password FOXESS_PASSWORD --pvoutput-api-key
               PVOUTPUT_API_KEY --pvoutput-system-id PVOUTPUT_SYSTEM_ID [--verbose]
               [--start-date START_DATE]

optional arguments:
  -h, --help            show this help message and exit
  --foxess-user FOXESS_USER
  --foxess-password FOXESS_PASSWORD
  --pvoutput-api-key PVOUTPUT_API_KEY
  --pvoutput-system-id PVOUTPUT_SYSTEM_ID
  --verbose             increase output verbosity
  --start-date START_DATE
                        Date when PV system was connected, YYYY-MM-DD format
```
